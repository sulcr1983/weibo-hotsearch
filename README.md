# 汽车行业舆情监控系统 V2.0 产品说明书
## 一、产品是什么？
汽车行业舆情监控系统 V2.0 是专为汽车行业从业者打造的「智能舆情管家」，不用再人工刷微博、翻新闻找行业动态，系统会自动帮你收集、筛选、整理全网关于目标汽车品牌的重要信息，每天/每周定时把核心内容推到你的飞书和邮箱，让你几分钟就能掌握行业和品牌的关键舆情。

## 二、核心功能：省心、精准、高效
### 1. 信息自动收，不漏任何重要动态
系统会全天候盯着两大核心信息源，不用你手动查找：
- **微博热搜**：实时抓取汽车相关的微博热点，不管是突发的品牌话题、用户热议的车型问题，都能第一时间捕捉；
- **11个主流新闻源**：覆盖专业汽车媒体、综合资讯平台，汇总权威的行业报道、政策解读、品牌动态。
所有信息每小时更新一次，保证你看到的都是最新内容，不会错过任何关键节点。

### 2. 智能筛信息，只看有价值的内容
不用再从海量信息里挑重点，系统会像专业分析师一样帮你过滤：
- **只盯你关注的品牌**：精准锁定小米汽车、鸿蒙智行、比亚迪、特斯拉等10个目标品牌，非目标品牌信息直接过滤，不浪费时间；
- **剔除无用信息**：通过智能算法去掉琐碎、无价值的负面吐槽，只保留对行业分析、品牌决策有帮助的内容；
- **避免重复看**：自动识别重复的新闻、话题，把相似内容归为一类，不用反复看相同信息。

### 3. 内容主动推，定时收现成报告
不用自己登系统查，核心信息会主动送上门：
- **每日日报（早9点）**：飞书+邮箱双渠道推送，汇总当天微博热搜和新闻里的品牌热点，重点品牌用彩色标签标注，一眼就能看清；
- **每周周报（周一早8点）**：不仅有一周热点汇总，还附带智能分析，按品牌分类整理明细，帮你梳理一周舆情趋势，省心又省力。

## 三、为什么选它？性能强，用着更顺手
### 1. 精准度拉满，不看无用信息
系统对品牌的识别精准到标题+正文双重校验，不会把无关品牌的内容混进来；智能算法筛选后，你看到的每一条信息都是和目标品牌相关、有实际参考价值的，告别信息杂乱。

### 2. 速度快，实时掌握动态
每小时更新一次全网信息，从信息采集到整理推送，全程自动化，比人工翻找快几十倍，不管是突发的品牌热点，还是最新的行业新闻，你都能第一时间知道。

### 3. 稳定不宕机，不用操心维护
系统采用独立模块运行设计，就算某一个功能模块临时出问题，也不影响整体使用；部署在稳定的服务器上，每天自动运行，不用专人维护，开机就能用。

### 4. 灵活又好用，适配你的需求
除了定时推送报告，还支持手动触发采集、生成报告，想临时查某段时间的品牌舆情也能快速搞定；报告内容结构化，品牌、事件、来源一目了然，不用花时间整理。

## 四、能帮你解决什么问题？
1. **省时间**：不用每天花几小时刷微博、翻新闻，几分钟看完推送报告就能掌握核心动态，节省80%以上的信息筛选时间；
2. **抓重点**：告别信息过载，只看和目标品牌相关、有价值的舆情，避免被无用信息干扰决策；
3. **知先机**：实时掌握行业热点、品牌突发动态，比竞品更早发现机会或风险；
4. **省人力**：不用安排专人做舆情整理，系统全自动完成，降低人力成本。

## 五、简单用，易上手
系统全程自动化运行，你只需要：
1. 确认关注的品牌清单（已预设10个主流品牌，可按需调整）；
2. 绑定接收报告的飞书账号和邮箱；
3. 每天9点、每周一8点查收报告即可，无需其他操作。

不管是品牌方、行业分析师还是从业者，不用懂技术，就能轻松用它掌握汽车行业的全量舆情动态！
